# Custom Adapter - Lockwood Oval (Australia/NZ)

**Type**: 3D Printable (Custom)
**Fits**: Lockwood oval cylinders
**Nuki Model**: Smart Lock Pro
**Status**: ⚠️ DESIGN NEEDED

---

## The Problem

Australia and New Zealand use **Lockwood oval cylinders** in some doors. Nuki doesn't make an official adapter for this.

## Opportunity

**Design a 3D printable adapter** for Lockwood oval cylinders.

### Measurements Needed

To design this adapter, measure:

1. **Outer diameter** of the oval cylinder body
2. **Inner bore** where the key goes
3. **Knob dimensions** (if knob style)
4. **Height** of the cylinder

### CAD Requirements

- Must interface with Nuki Pro mounting socket
- Must grip Lockwood oval knob securely
- Recommend PETG for durability

## Alternative Solutions

If you can't print an adapter:

1. **Replace the cylinder** with euro profile
2. **Replace the lock** with euro mortise lock
3. **Use alternative smart lock** (August, Level)
