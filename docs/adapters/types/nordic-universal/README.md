# Nordic Universal Cylinder

**Type**: Replacement cylinder (not adapter)
**Fits**: Scandinavian oval/round cylinders
**Nuki Model**: Smart Lock Ultra Nordics
**Included**: Yes - with Ultra Nordics bundle

---

## How It Works

Unlike other Nuki models that use **adapters** over existing cylinders, the Nordic solution **replaces your entire cylinder**.

```
┌─────────────────────────────────────────────────────────┐
│         STANDARD EU vs NORDIC APPROACH                   │
├─────────────────────────────────────────────────────────┤
│                                                          │
│   STANDARD (EU/US)            NORDIC                    │
│                                                          │
│   ┌─────────────┐            ┌─────────────┐           │
│   │    NUKI     │            │    NUKI     │           │
│   │             │            │    ULTRA    │           │
│   │  [ADAPTER]  │            │             │           │
│   │      │      │            │             │           │
│   │      ▼      │            │      │      │           │
│   │  [EXISTING  │            │      ▼      │           │
│   │  CYLINDER]  │            │ [UNIVERSAL  │           │
│   └─────────────┘            │  CYLINDER]  │           │
│                              └─────────────┘           │
│                                                          │
│   Adapter over                Replaces                  │
│   existing                    existing                  │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

## Included With

The Universal Cylinder comes **bundled** with Smart Lock Ultra Nordics. Not sold separately.

**Bundle price**: ~€349

---

## Installation

1. Remove existing Scandinavian cylinder
2. Measure door thickness
3. Install Universal Cylinder
4. Mount Nuki Ultra over cylinder
5. Configure via app

---

## Security

The Universal Cylinder meets:
- **SS 3522** (Swedish standard)
- **EN 1303** (European standard)
