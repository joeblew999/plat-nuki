## Image Resolution Variants

Nuki CDN supports different resolutions by changing the `@1920` in the URL:

```
@3200 - High resolution (3200px)
@1920 - Standard (1920px) ← We use this
@1366 - Medium (1366px)
@960  - Small (960px)
```

Example:
```
https://media.crystallize.com/nuki-production/24/12/3/45/@3200/nuki_knob-adapter2.webp
```
