## Summary

The UK is an official Nuki market, but has **two common lock types** - modern euro cylinders (compatible) and traditional night latches/rim locks (not compatible). Check your door before buying.

---

## Lock Types in UK

### Type 1: Euro Cylinder (Modern Doors) ✅

Common in:
- UPVC doors (most common modern door type)
- Composite doors
- Modern wooden doors
- Apartment buildings

**Nuki Compatible**: Yes - Smart Lock Pro

### Type 2: Night Latch / Yale Lock (Traditional) ❌

Common in:
- Victorian/Edwardian terrace houses
- Older properties
- Some flats with original doors

**Nuki Compatible**: No - this is a rim lock

## How to Identify Your Lock

### Euro Cylinder Signs:
- Drop-shaped keyhole
- Key inserts horizontally
- Handle below the lock
- Lift handle to engage multipoint lock

### Night Latch Signs:
- Round keyhole
- Surface-mounted box on inside of door
- Snib (small thumb turn) on the box
- "Yale" branding common

## Insurance Considerations (Important!)

UK home insurers often require **BS 3621** compliant locks.

- Nuki mounts OVER your existing lock
- Your underlying lock should still be BS 3621
- Keep the original cylinder functional
- Some insurers may require notification

**Check with your insurer before installing.**

## Installation Considerations

### UPVC Doors

- Most common modern UK door
- Euro cylinder standard
- Multipoint locking (lift handle to engage)
- Nuki works well

### Composite Doors

- Similar to UPVC
- Euro cylinder standard
- Check door thickness

### Traditional Wooden Doors with Night Latch

**Options if you have a night latch:**
1. Add a euro cylinder mortise lock (requires door modification)
2. Use an alternative smart lock designed for rim locks
3. Check if door has a secondary deadbolt that could work

## Rental Properties

- **Check tenancy agreement** before installing
- Nuki is non-destructive (no drilling)
- Keep original hardware to reinstall when leaving
- Inform landlord as courtesy
