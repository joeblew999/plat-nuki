# Indonesia - Nuki Compatibility Guide

**Status**: ⚠️ UNOFFICIAL (Mixed market)
**Nuki Model**: Smart Lock Pro (EU) for euro cylinders
**Confidence**: Medium

---

## Summary

Indonesia has a **mixed lock market** - modern condos and commercial buildings often use euro cylinders (compatible), but houses vary widely. Assess on case-by-case basis.

## Installation Considerations

### Apartments

- **Building management**: Check rules
- **Fire doors**: Don't modify door itself
- Common in Jakarta high-rises

### Houses

- **Gate vs door**: Nuki for main door, not gate
- **Grille doors**: Common - Nuki goes on main door
- **Security**: Consider full security system
