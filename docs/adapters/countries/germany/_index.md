## Lock Types in Germany

### Standard: Euro Profile Cylinder (DIN 18252)

Germany uses **euro profile cylinders** almost exclusively. This is what Nuki was designed for.

![Cylinder Types](../../diagrams/cylinder-types.webp)

## Installation Considerations

### Apartment Buildings (Mietwohnung)

- **Landlord permission**: Check your rental contract
- **No drilling**: Nuki doesn't require drilling the door
- **Key return**: Keep original cylinder, reinstall when moving out

### Own Property (Eigentum)

- Full freedom to install
- Consider Nuki Door Sensor for activity logging
- Bridge for remote access

### Fire Doors (Brandschutztüren)

- Nuki mounts on inside, doesn't affect fire rating
- Don't modify the door itself

---

## Insurance (Hausratversicherung)

- Most German insurers accept smart locks
- Keep manual key access as backup
- Document installation for insurance purposes
