# Hong Kong - Nuki Compatibility Guide

**Status**: ⚠️ UNOFFICIAL (Ships internationally)
**Nuki Model**: Smart Lock Pro (EU)
**Confidence**: High

---

## Summary

Hong Kong is **not an official Nuki market**, but euro profile cylinders are standard in most residential and commercial buildings. Smart Lock Pro should work well.

---

## Lock Types in Hong Kong

### High-Rise Residential (Most Common)

| Building Type | Lock Type | Nuki Compatible? |
|---------------|-----------|------------------|
| **Private residential** | Euro cylinder mortise | ✅ Yes |
| **Public housing (公屋)** | Euro cylinder | ✅ Yes |
| **HOS flats** | Euro cylinder | ✅ Yes |
| **Village houses** | Mixed | ⚠️ Check |

### Commercial

| Type | Lock Type | Nuki Compatible? |
|------|-----------|------------------|
| **Office buildings** | Euro cylinder / access control | ✅ Yes (standalone doors) |
| **Retail** | Euro cylinder | ✅ Yes |

## Installation Considerations

### Private Flats

- **DMC/OC rules**: Check with Deed of Mutual Covenant or Owners' Corporation
- Most installations are on unit doors, not common areas
- Grille doors common - Nuki goes on main door, not grille

### Public Housing

- **Housing Authority rules**: Check regulations
- May need approval for modifications
- Keep original hardware
