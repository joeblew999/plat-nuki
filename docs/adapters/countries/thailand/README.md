# Thailand - Nuki Compatibility Guide

**Status**: ⚠️ UNOFFICIAL (Ships internationally)
**Nuki Model**: Smart Lock Pro (EU)
**Confidence**: Medium-High

---

## Summary

Thailand is **not an official Nuki market**, but condominiums and modern residential buildings commonly use euro profile cylinders. Good potential for Nuki Pro.

---

## Lock Types in Thailand

### Bangkok & Major Cities

| Property Type | Lock Type | Nuki Compatible? |
|---------------|-----------|------------------|

| **Serviced apartments** | Euro cylinder | ✅ Yes |

## Regional Considerations

### Bangkok

- High-rise condos: Euro cylinders standard
- Good smart home adoption
- Reliable power and internet

### Phuket / Samui / Resort Areas

- Villas often have euro cylinders
- Good for rental management (Airbnb)
- Consider humidity near beach

### Chiang Mai / Regional Cities

- Mix of old and new
- Newer condos: Euro cylinders
- Older buildings: May vary

---

## Installation Considerations

### Condominiums

- **Juristic person rules**: Check with building management
- Common in Bangkok for owners to modify
- Keep original hardware

### Rental Properties

- Great use case for Nuki (remote access for guests)
- Check landlord agreement
- Consider subscription for rental platform integration
